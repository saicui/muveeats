Beta v.1 Landing Brief

MuveEats — 動いて食べて、記録する。

食事・運動・体組成を一つにまとめる個人ヘルスハブ。Claude Design へのランディング制作ハンドオフ用ブリーフ。

1. ポジショニング

One-liner: 「食事」「運動」「体組成」をひとつのアプリでシンプルに記録する。AI が記録の手間を肩代わりし、直近の傾向からあなた専用のフィードバックを返す。

"3つのアプリで管理するのをやめた。書く・撮る・選ぶ、どれでも 5 秒。"

このプロダクトの立ち位置

  • 記録の分散をやめる: 食事記録アプリ・運動記録アプリ・体組成アプリを別々に開かず、1 つにまとめる。
  • AI を「相談相手」ではなく「入力補助」に: 写真 → 自動で栄養計算、体組成計の表示画面 → 自動で数値抽出。
  • テンプレ × クイック追加: 毎日同じ朝食 / 胸の日 などをワンタップ。継続のための摩擦を最小化。

2. 強み(ランディングで打ち出したい順)

#強み具体
1シンプルに 3 つを 1 つに食事 / 運動 (筋トレ + 有酸素) / 体組成を 1 画面のホームで一望。
2AI による記録自動化食事写真 → AI が栄養素を推定。体組成計の写真 → 数値 OCR。補足文 (「ライス少なめ」等) で精度向上。
3AI フィードバック直近 7 日の食事 + 運動 + 体組成 + 目標を集約 → 良い点 / 改善 / 明日のアクションを 1 タップで生成。
4テンプレで継続摩擦ゼロ「毎朝のオートミール」「胸の日」を 1 タップ記録。後日記録もサポート。
5主要チェーン店 + 多数の品目国内の主要外食・カフェ・牛丼・ピザ・コンビニなどの栄養情報を内蔵 (オフライン検索)。
6純粋にWebインストール不要。スマホでもデスクトップでも同じ URL。ホーム画面追加で PWA 風に。

3. ターゲット

プライマリ

  • 20–40 代の自炊 × ジム勢。減量 / 筋肥大の目標がある人。
  • 有料の食事記録アプリに課金するまでではない、でも体重と食事は記録したい層。

セカンダリ

  • 外食 / コンビニ多めで PFC を雑に把握したいビジネスパーソン。
  • パーソナルトレーナーに食事報告したいクライアント (CSV エクスポート想定)。

6. アプリの現在の画面構成 (Beta v.1)

ホーム /

  • INTAKE / BURN サマリ (今日)
  • エネルギーバランス (摂取 vs 消費)

  • AI フィードバック (オンデマンド生成)
  • 直近 7 日ドットマトリクス
  • クイック追加 (食事/筋トレ/有酸素/体組成)
  • 食事テンプレのワンタップ追加
  • 最近の記録 8 件

食事 /meals/new

  • 後日記録の日時指定 (任意)
  • テンプレから記録 (任意)
  • 3 モード: 検索 / 写真 / 手動
  • 検索: 32 チェーン × 約 2000 品目
  • 写真: 補足文 (200 字) で精度向上 / AI 解析
  • 手動: 名前 + PFC + kcal を直接入力
  • タグ自動推論 (高タンパク / 脂質高め / 朝食 等)

筋トレ /workouts/new

  • セッションタイマー
  • 後日記録の日時指定
  • テンプレから開始 (胸の日 / 脚の日 等)
  • 種目を追加 → セット (重量×回数) を入力
  • 81 種目 + 部位フィルタ + カスタム種目
  • 過去の重量を参考表示
  • セッション完了で MET 法による消費 kcal 概算

有酸素 /cardio/new

  • 後日記録の終了日時指定
  • 種目 (ラン/ウォーク/バイク/その他) + 強度 (低/中/高)
  • 時間 + 距離 → ペース自動算出
  • 体重と強度から消費 kcal を MET 法で概算

体組成 /body, /body/new

  • 体重 / 体脂肪率 / 筋肉量 / 内臓脂肪 / BMR / BMI
  • 体組成計の写真 → 数値 OCR (AI)
  • 体重・体脂肪率の sparkline
  • 記録履歴 20 件

履歴 /history

  • 食事 + 運動 + 体組成を時系列で統合表示
  • 詳細シート (タップで開く)
  • 削除可能
  • 直近 200 件 (食事) / 100 件 (運動 / 体組成)

テンプレ管理

  • 食事テンプレ /templates: 定常分 + 当日スキップ
  • 運動テンプレ /workouts/templates: 種目 + セット (重量×回数) のひな型

設定 /settings

  • プロフィール (性別 / 年齢 / 身長 / 活動量 / 目標)
  • 目標 kcal + PFC の自動算出 (Mifflin-St Jeor)
  • AI に目標相談 (チャット形式)
  • テーマ切替 (Light / Dark / System)
  • テンプレ管理への導線
  • サインアウト
  • 記録を全て削除 (危険ゾーン)
