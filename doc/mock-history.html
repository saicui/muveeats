💡 機能のポイント / 検討事項

  • 日付グルーピング: 同じ日の食事はまとめて、その日の合計 kcal を表示。スクロールで時系列に振り返る。
  • 期間タブ: 今日 / 7日 / 30日 / すべて で素早く切替。サマリーとミニグラフは選択期間に追従。
  • ミニグラフ: 7日の日別 kcal を棒グラフで。目標オーバーは黄色、今日は濃緑で強調。
  • サマリー4カード: 期間合計 / 1日平均 / タンパク質平均 / ストリーク。目標値は別途設定画面が必要（今回スコープ外）。
  • 検索: 料理名の部分一致 (ilike)。種別フィルタ（写真 / 手動）も用意。
  • 詳細表示: モーダル vs 別ページ どちらが良いか？ モバイル中心ならフルスクリーンモーダルが操作しやすい。
  • 編集 / 削除: 詳細から行う想定。編集は /log の再利用、削除は確認ダイアログ。
  • 写真サムネ: 現状 photo_url 未保存。サムネ表示するには Supabase Storage への保存が前提（フェーズ2）。今回は絵文字プレースホルダで運用するか、サムネを諦めるか。
  • ストリーク: 「連続で1食以上記録した日数」を表示。モチベ用。計算ロジックは要相談。
