v7 で変えたこと（v6 からの差分）

  • Screen 4 (有酸素) の種目セルを修正: ラベルを英語化 (RUN / WALK / BIKE / OTHER)、4 種目それぞれ専用アイコンに差し替え (#i-run / #i-walk / #i-bike / #i-activity)。.qa セルに flex / column / center を加え、アイコン + ラベルの縦位置を常に中央揃えに。
  • フィールド入力欄の幅を可変に: .field-group .field input/select の width: 130px 固定をやめ、min-width: 100px; max-width: 200–220px; width: auto; に。<select> は text-align-last: right で右寄せ表示を維持しつつ、長い option ラベルが見切れないように。
  • ドロップダウン文言を簡潔化: Screen 4 強度を「中 (やや息が上がる)」→「中」、Screen 7 単位を「メートル法 (kg / km)」→「メートル法」、活動レベル / 目標も括弧書き説明を削減。

v6 で変えたこと（v5 からの差分）

  • Pastel テーマ撤廃: Light との差が小さく、3テーマ運用のコストに見合わないと判断。data-theme="pastel" ブロックと Settings のスウォッチを削除。トークンは Light / Dark の 2軸のみに。
  • 体組成連携の概念を撤廃: Apple Health / 体組成計の自動同期は今後フェーズでも前提にしない方針へ。Settings から「Apple Health 連携」行を削除、Screen 10 から「写真 OCR」ボタンを削除、各履歴のソースを 体組成計 → 手動記録 に。記録はすべて自分で入力するアプリとして言語を統一。
  • Meal Detail を編集可能に: 料理名 / 時刻 / kcal / PFC / タグをすべてシート内で編集できるように再設計。footer は [削除] [キャンセル] [保存] の3ボタン構成。AI メモは引き続き読み取り専用。
  • v5 で確認したい点への合意: AI 相談ボックスは最上部のまま / Exercise Picker の PIN は採用継続。
